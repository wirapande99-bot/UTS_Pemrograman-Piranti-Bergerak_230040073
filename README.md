# Aplikasi Manajemen Data Siswa (UTS) - Sage Green Edition

Aplikasi Android native yang dikembangkan menggunakan Java untuk tugas Ujian Tengah Semester (UTS) Pemrograman Piranti Bergerak. Aplikasi ini memungkinkan pengelolaan data siswa dengan penyimpanan lokal SQLite dan antarmuka bertema **Sage Green**.

## Fitur Utama
- **Login System**: Akses aman dengan username/password (`admin`/`admin`).
- **CRUD Siswa**: Menambah, menampilkan, dan menghapus data siswa.
- **Data Field**: NIM, Nama, Jurusan, dan Nilai.
- **Tampilan Modern**: Menggunakan Material Design, RecyclerView, dan CardView dengan tema warna Hijau Sage yang estetis.
- **Penyimpanan Lokal**: Menggunakan SQLite agar data tetap tersimpan secara permanen.

## Cara Instalasi & Menjalankan
1. Clone repository ini.
2. Buka di **Android Studio**.
3. Pastikan SDK Android terinstal (Min SDK 24).
4. Run pada Emulator atau Perangkat Fisik.
